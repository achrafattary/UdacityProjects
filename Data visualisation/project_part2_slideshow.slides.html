FordGoBike data : looking whether different type of users have different type of usages habits¶

by Attary achraf¶

Investigation Overview¶

the goal of this project is to study whether certain variable like age and user type affect how the users uses this service

Dataset Overview¶

this data set contains rides from february of 2019,the dataset have many informations both about the rides(duration,time_start,end_time,location_start ect) and about the users (age,user_type,ect)

Visualization 1¶

this visual show that the vast majority of the user base is subscribed,the retention rate must be very hight and the user satisfied

Visualization 2¶

here we can see that whether the user is subscribed or not,they still use the service at the same hour every day with a peak at 8 in the morning and 18 probably because workers use the service to get home

Visualization 3¶

in this visual we can see that users folow the same pattern form the duration of their ride,especially the users in the age of work,this also make us think that most users use the service to move from their jobs to their work and vise-versa

Visualization 4¶

this visual also shows that gender does not make a difference in the pattern of usage of the service
